# Documentação ERP Sabor da Terra

Esta pasta contém a documentação do sistema ERP Sabor da Terra.

## Arquivos

- **DOCUMENTACAO_ERP_SABOR_DA_TERRA.md** - Documentação geral do sistema
- **DOCUMENTACAO_ERP_SABOR_DA_TERRA.pdf** - Documentação em formato PDF
- **PROPOSTA_MENU_NAVEGACAO.md** - Proposta de menu e navegação (layout, UX, itens admin)
- **gerar_pdf.py** - Script para regenerar o PDF a partir do Markdown

## Regenerar o PDF

Para atualizar o PDF após alterações no Markdown:

```bash
pip install markdown-pdf
python docs/gerar_pdf.py
```
